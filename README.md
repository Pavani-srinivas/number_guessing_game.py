# Number Guessing Game

## About the Project
This is a simple Python game where the user has to guess a number chosen by the system.  
The user is given limited attempts, and the score is updated when the guess is correct.

This project was created to practice Python basics and logical thinking.

---

## Features
- Random number generation
- Limited attempts for guessing
- Score tracking
- Input validation
- Option to play again

---

## How It Works
- The program selects a random number between 1 and 10.
- The user gets 3 chances to guess the number.
- After each guess, the program gives a hint (too high or too low).
- If the guess is correct, the score increases.
- The user can choose to replay or exit the game.

- Exception handling
